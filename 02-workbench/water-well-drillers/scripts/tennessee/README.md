# Tennessee Water Well Contractors Ingestion Runbook

This directory handles harvesting, parsing, and cleaning water well drillers and pump installers for the State of Tennessee.

---

## Data Source

*   **Agency:** Tennessee Department of Environment & Conservation (TDEC)
*   **Source Format:** Oracle APEX Web App Database
*   **Original URL:** [TDEC Report on Licensed Drillers](https://dataviewers.tdec.tn.gov/dataviewers/f?p=2005:39918)
*   **Target Classes:** Classes W (Water Well Driller), P (Pump Installer), T (Treatment Installer), M (Monitoring Well), G (Geothermal), C (Casing).

---

## Technical Extraction Process & Quirks

The Tennessee database is managed via an Oracle APEX interactive report application. It has the following constraints:

1.  **Headless 403 Blocking:** Simple HTTP clients (e.g., `requests`, `urllib`) receive a strict **403 Forbidden** response when requesting pages on the `dataviewers.tdec.tn.gov` domain. 
2.  **Browser Emulation:** To scrape or download records from this portal, Playwright or another browser-based automation library must be used to request pages with valid Chrome User-Agent strings and hydrated headers.
3.  **APEX Report Interaction:** Once on the page, the user must navigate through Oracle APEX interactive selectors (e.g., finding the "Actions" dropdown or looking for direct download URLs generated by the APEX engine).
